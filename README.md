# Animated Navigation Menu Bar

A pure CSS/HTML animated navigation component with glassmorphism design and smooth sliding transitions.

> **Note:** This is a CSS practice project demonstrating animation techniques. No actual page routing included.

![Preview](screenshot.png)

## Live Demo
[Click here to view](https://Mehdi-dev-sudo.github.io/Animated-Login-Form/)

## Features
- Animated sliding indicator
- Ripple effect on click
- Glassmorphism UI design
- 5 gradient color themes per section
- Fully responsive (desktop to mobile)
- Pure CSS - no JavaScript required

## Tech Stack
HTML5 · CSS3 · Font Awesome

## How It Works
Uses hidden radio inputs and CSS `:checked` pseudo-class to trigger animations without JavaScript. The slider position changes based on which radio button is selected.
